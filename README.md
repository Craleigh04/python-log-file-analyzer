# Python Log File Analyzer

This project analyzes log file data to extract useful insights such as unique event messages, filtered IP addresses, and event counts per node.

## Overview
The program reads a structured log file and allows users to interact with the data through a menu-driven interface.

Users can:
- View unique event messages
- Search for IP addresses by octet value
- Count events per node

## Features
- File parsing using string manipulation
- Data extraction from structured logs
- Dictionary-based counting
- Modular function design
- Menu-driven user interface
- Exception handling with error logging

## Skills Demonstrated
- Python programming
- File I/O and data processing
- String parsing and manipulation
- Lists and dictionaries
- Modular programming
- Error handling (try/except)
